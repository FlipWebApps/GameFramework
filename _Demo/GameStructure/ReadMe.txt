This demo provides a introduction to some of the features available within the GameStructure section of Game Framework. For
further information we recommend going through the online tutorials and documentation.

To run this demo perform the following steps:
	1. Open build settings from "Top Menu->File->Build Settings".
	2. Drag all scenes to "Scenes In Build" in the "Build Settings" window.
	3. Open and run the "GameStructure-Title" scene

For more details on Game Framework please visit: http://www.flipwebapps.com/game-framework/ 
For tutorials and information on getting started visit: http://www.flipwebapps.com/game-framework/tutorials/
